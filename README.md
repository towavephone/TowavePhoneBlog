## 使用

### Yarn

``` bash
yarn
// 因为 node_modules 文件夹里面有改动
git checkout .
yarn dev
git add .
// 输入的提交信息必须为英文
yarn cm
yarn prod
yarn deploy
```

### npm

``` bash
npm install
git checkout .
npm run dev
git add .
npm run cm
npm run prod
npm run deploy
```
